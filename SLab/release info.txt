=======================================================
 Main Folder Contents
=======================================================

This is the main folder of the SLab project

It contains the following subfolders:

Boards : Source and binnary for the supported 
         Hardware boards

Code : Forder where slab.py is located
       Also contains some other non standard modules

Jupyter : Jupyter notebooks for the tutorial and
          the system documentation

Scripts : Operating System scripts

Setup Docs : Documentation for SLab installation

Jupyter.bat : Windows script to launch SLab 

Jupyter Custom.bat : Windows launch script with custom Python paths

Jupyter.sh : Linux script to launch SLab

=======================================================
 Launching SLab
=======================================================

Information on how to install and lauch SLab are in the
Setup Docs folders.
