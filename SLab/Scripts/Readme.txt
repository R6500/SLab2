This folder contains several scripts

Anaconda_Scripts_CMD.bat
Open a console in the scripts folder of Anaconda
Requires Anaconda to be installed on /Anaconda_Win64

Anaconda_SLab_Jupyter.bat
Run Jupyter in the SLab/Jupyter folder
Requires Anaconda to be installed on /Anaconda_Win64
Requires SLab to be on /SLab

Clean_SLab.bat
Cleans the SLab folders
  Removes list (.lst) and object (.o) files for the firmware
  Removes .pyc files
  Removes __pycache__
  Remove examples and Tests
  Remove calibration data
  Remove notebook checkpoints
  Remove private folders
  Clears the SLab/Jupyter/Files folder
